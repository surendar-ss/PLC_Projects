# PLC-Projects
PLC Ladder Logic programs built  in CODESYS - Motor Control,  Self Latching, TON Timer |  Industrial Automation Learning Journey

## Programs Built

### Day 1 — Motor Start Stop
- Built in CODESYS Ladder Logic
- Start button ON → Motor runs
- Stop button ON → Motor stops
- Zero errors in simulation

### Day 2 — Self Latching Circuit
- Press start ONCE → Motor stays ON
- Press stop → Motor turns OFF
- PLC addressing: I0.0 I0.1 Q0.0 M0.0
- Real industrial motor control
